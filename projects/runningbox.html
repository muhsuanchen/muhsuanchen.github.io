Running Box

NTOU  |  Junior Spring
3D Game Programming and Design

在這撲滿遠古石板路的森林中，遠方依稀傳來尖嘯的狼嚎，迷途的孩子與飢餓的狼，為了離開這個森林 / 捕獲今晚的佳餚，開始了漫長的奔馳旅程!

Running Box 是款角色扮演動作遊戲，主要遊戲架構以 Temple Run 為發想，並結合了多款經典遊戲設定，諸如首頁的旋轉地圖與角色選單的展現，以及單機高分記錄。遊戲中人物將會自動向前奔跑，路途中將遇到多種各具特色的陷阱，玩家需要展現其反應能力來避開它們，並且盡量多的獲取金幣，以取得更高的分數!

- 本遊戲以 OpenGL 與 C 為主要語言，並使用 OpenAL 產生音效。開發過程 使用 Microsoft Visual C++ 2010 Express 做為開發環境。
- 使用 Finite-state machine 來 控制每一階段的動作，使角色動作切換更為流暢。
- 使用多種資料結構來儲存地圖資訊，讓遊戲地圖達到無限延伸的效果。
- 遊戲畫面以 Skybox 技術製作背景，並使用 Billboard 概念製作場景內容。
