Computer Graphics with OpenGL

小畫家作業，透過 OpenGL 實作 2D 上的繪圖功能，以熟悉 OpenGL 繪圖效果基礎概念。

- 提供當前筆刷預覽。
- 提供多種基礎圖形、字元輸入功能，且可切換圖形輸入與筆刷效果。
- 提供多種顏色與混色功能。
- 提供暫存、載入、Undo與清空功能。
- 提供 bmp 圖檔存檔功能。
- 提供圖層 Blending 功能。

Note:
- 橡皮擦功能使用 blending 來與空圖層相互疊加，實際效果因電腦本身 library function 而定。
