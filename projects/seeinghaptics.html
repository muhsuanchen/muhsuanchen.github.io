SeeingHaptics

Visualization for Preview, Examination, and Display of Haptic Designs.
Mu-Hsuan Chen, Wen-Chi Ho, Li-Wei Chan, Bing-Yu Chen.

NTU  |  Jan. 2017 -
Master Thesis. Advisor: Prof. Li-Wei Chan, Prof. Bing-Yu Chen.

Rendering haptic feedback in virtual reality is a common approach to enhance the immersion of the contents. However, current editing tools for developing virtual environment only allows developers to access haptic feedbacks by touching the actuators, which makes it difficult to overview the design effectively.

This paper introduces SeeingHaptics, an authoring tool which visualizes properties of haptic feedbacks in 3D scenes. The active area of a certain feedback is displayed with a mesh (Figure: Teaser a), while a 2D icon attaching by the mesh indicates the type of haptic sensation (Figure: Teaser b). Within the virtual environment, the haptic feedbacks generated by virtual contents act as Haptic Source components on Unity interface. At the same time, Haptic Proxy, a hardware prototype providing corespondent physical haptic feedback allows developers to examine the design interactively by touching and experiencing. The evaluation showed that SeeingHaptics is a learnable, flexible, and potential authoring tool for rapid design of various haptic effects.
