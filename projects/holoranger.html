全息射手 HoloRanger

A Multi-Layer Panorama Mobile Game Using Around-Body Interactions.
Shan-Yuan Teng, Mu-Hsuan Chen, Yung-Ta Lin.

NTU  |  Jan. 2017 - May. 2017

以 Unity3D 開發之第三人稱射擊遊戲，發行 Android 與 iOS 雙平台。

My Contribution:
- 擔任主力開發。 - 遊戲內技能系統、遊戲物理機制、人物移動與外力影響系統、遊戲進場離場流程、遊戲結算舞台場景與流程開發。
- 多人線上遊戲流程與 Command Handshake 機制制定與開發。
- 獨立場景製作、美術資源打包處理、美術資源串接之經驗。
- 資料表格系統設計、自動分析 Log 之網頁設計。
- 與 Server、Art、PM 皆有密切合作之經驗。
